# MovieFinder

MovieFinder is an Web Application you can use to search for movies and look at their details.

It has been developed using [Typescript](https://www.typescriptlang.org/) and [ReactJS](https://reactjs.org/).

## Prerequisites

`npm`

## How to run

1. `npm install`
2. `npm run dev`

To run the tests:

1. `npm install`
2. `npm test`
